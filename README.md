# slot-machine-
- spin the slots, place your bets, match symbols, and try to increase your balance!


 # How the Game Works
 ```
you start with a balance of $100
choose how much you want to bet each round
the slot machine spins 3 random symbols
if all three symbols match, you win based on the symbol value
you can keep playing until you run out of money or choose to quit

```
